# Тест-кейсы: Тестирование GitHub REST API

## Объект тестирования: GitHub REST API v3
**Уровень доступа:** Аутентифицированный пользователь (Personal Access Token с правами `read:user` и `public_repo`).
**Базовый URL:** `https://api.github.com`

---

### Кейс 1: Успешная аутентификация и получение профиля пользователя
- **Эндпоинт:** `GET /user`
- **Предусловие:** Имеется валидный Personal Access Token с правом `read:user`.
- **Шаги:**
    1.  Установить в заголовке `Authorization` значение `Bearer <token>`.
    2.  Отправить GET-запрос.
- **Ожидаемый результат:**
    - Код статуса: `200 OK`.
    - Тело ответа содержит корректный JSON-объект профиля.
    - Объект содержит поля: `login`, `id`, `type`.
    - Значение поля `login` соответствует логину владельца токена.
    - Значение поля `type` равно `"User"`.
- **Фактический результат:** Сервер вернул `200 OK`. В ответе присутствуют все ожидаемые поля. Поле `login` содержит `"sv-medyulyanov-exe"`, поле `type` равно `"User"`.
- **Статус:** Pass ✅
- **Автоматическая проверка (Postman Test):** Пройдены проверки статуса, наличия обязательных полей и соответствия логина.

### Кейс 2: Получение списка репозиториев текущего пользователя
- **Эндпоинт:** `GET /user/repos`
- **Предусловие:** Пользователь имеет как минимум один публичный репозиторий.
- **Шаги:**
    1.  Установить заголовок авторизации.
    2.  Отправить GET-запрос.
- **Ожидаемый результат:**
    - Код статуса: `200 OK`.
    - Тело ответа — массив объектов (репозиториев).
    - Каждый объект в массиве содержит поля: `id`, `name`, `full_name`, `private`.
    - В массиве присутствует репозиторий `"api-testing-sandbox"`.
- **Фактический результат:** Сервер вернул `200 OK`. Ответ является непустым массивом объектов. Найден репозиторий с `name: "api-testing-sandbox"`.
- **Статус:** Pass ✅
- **Автоматическая проверка (Postman Test):** Пройдены проверки статуса, типа ответа (массив) и наличия ожидаемого репозитория в списке.

- **Предусловие:** Репозиторий `api-testing-sandbox` существует и является публичным.

- **Фактический результат:** Сервер вернул `200 OK`. Ответ содержит все ожидаемые поля. `name: "api-testing-sandbox"`, `private: false`.
