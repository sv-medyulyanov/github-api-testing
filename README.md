# GitHub API Testing Project

Практический проект по тестированию REST API GitHub с авторизацией.

## Цель проекта
- Освоить работу с OAuth-токенами (Personal Access Token).
- Протестировать ключевые эндпоинты GitHub API.
- Настроить автоматические проверки в Postman.
- Создать портфолио с реальными примерами.

## Что протестировано
1.  **Аутентификация** — настройка Bearer Token.
2.  **Профиль пользователя** — эндпоинт `GET /user`.
3.  **Репозитории пользователя** — эндпоинт `GET /user/repos`.

## Структура
- `guide/` — пошаговые инструкции.
- `screenshots/` — доказательства выполнения тестов.
- `postman_collection/` — экспортированная коллекция Postman.

## Технологии и инструменты
- GitHub REST API v3
- Postman (переменные окружения, тесты)
- Markdown для документации
